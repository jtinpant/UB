## 🚀 Overview
**SMOKER USERBOT** is a unique repository designed for dynamic hosting. Unlike traditional userbots, this allows any user to host their own ID simply by interacting with your main bot. No complicated deployment is needed for the end-user—just send your number and start!

## ✨ Key Features
* **Multi-Client Hosting:** Host unlimited user IDs simultaneously.
* **Dynamic Login:** Users can login/logout via a simple bot button interface.
* **Raid Module:** Includes Standard Raid, Love Raid (MRAID), Shayari Raid (SRAID), and ABCD Raid (CRAID).
* **Spam Module:** High-speed Text Spam, Media/GIF Spam, and PornSpam with group protection.
* **Hang Command:** Specialized lag/hanging messages for fun purposes.
* **Sudo Control:** The owner maintains full control over all hosted sessions.

## 🛠️ Setup Instructions

### 1. Requirements
* Python 3.9 or higher.
* `API_ID` and `API_HASH` from [my.telegram.org](https://my.telegram.org).
* A Bot Token from [@BotFather](https://t.me/BotFather).

### 2. Configuration
Edit your `config.py` with the following essential values:
* `API_ID`: Your Telegram API ID.
* `API_HASH`: Your Telegram API Hash.
* `BOT_TOKEN`: The token for your main hosting bot.
* `OWNER_ID`: Your Telegram User ID.

### 3. Installation
```bash
git clone [https://github.com/lavish-pro/SMOKER-USERBOT](https://github.com/lavish-pro/SMOKER-USERBOT)
cd SMOKER-USERBOT

python main.py
